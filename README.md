# DYML-wordcloud

## 项目全名Douyin_My_Love_WORLDCLOUD(抖音喜欢词云图)



# 使用方法

1. 微软应用商店安装mitmproxy
2. 安装相关依赖 `pip install -r requirements.txt`
3. 运行 main.py
4. 根据提示另开power shell或者cmd启动代理
5. 根据提示到浏览器打开主页喜欢列表并滑动至最低端
6. 回到main.py进行相关输入

# 项目原理

利用mitmdunp进行抓包获取抖音喜欢的json数据,使用wordcloud等库对数据进行处理
